Key Technical Features

Component Structure: - Carousel (main container) - CarouselViewport (visible area) - CarouselTrack (slide container with transforms) - CarouselSlide (individual slides) - NavigationButton (prev/next controls) - PaginationBullets (dot indicators) State Management: - Current slide index tracking - Animation state management - Touch/swipe gesture handling - Responsive breakpoint detection Performance Optimizations: - GPU-accelerated CSS transforms - will-change property for smooth animations - Backface-visibility to prevent flickering - Efficient re-render patterns Accessibility Features: - ARIA attributes for screen readers - Keyboard navigation support - Focus management - Reduced motion support

Responsive Behavior

The carousel automatically adjusts to different screen sizes with the following breakpoints:

320px: 1 slide per view, 10px spacing 640px: 1.2 slides per view, 15px spacing 768px: 1.5 slides per view, 20px spacing 1024px: 2 slides per view, 25px spacing 1280px: 2.5 slides per view, 30px spacing 1536px: 3 slides per view, 35px spacing
